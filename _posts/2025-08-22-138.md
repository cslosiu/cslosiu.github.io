有個新的api返回的json怎樣都出錯，最後找到是它第一位有個「零長不斷行空格」的神奇字符 。用以下方式清理後就可以了 。
public static String cleanup(String networkData) {
  return networkData.replace((char)65279, ' '); // zero length non breaking space.
}
